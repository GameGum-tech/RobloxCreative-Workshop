# みんなでアスレチックゲーム（Obby）を作ろう！ワークショップガイド

## 1. 今日のゴール
Roblox Studio（ロブロックス スタジオ）を使って、チームのみんなで協力して一つの「巨大アスレチックコース」を完成させることが目標です。自分だけの面白い仕掛けを作って、友達を驚かせましょう！

## 2. 画面の動かし方（カメラ操作）
ゲームを作るためには、自由に世界を見渡す必要があります。
* **W・A・S・D キー**: 前後左右に移動します。
* **マウスの右ドラッグ**: カメラの向き（視点）をぐりぐりと回します。
* **マウスホイール**: ズームイン・ズームアウトします。

## 3. パーツのいじり方（移動・サイズ・回転）
ブロック（パーツ）を自由自在に操るための4つの道具を使います。
* **選択（Select）**: パーツをクリックして選びます。
* **移動（Move）**: 矢印を引っ張って、好きな場所に動かします。
* **サイズ（Scale）**: 丸い玉を引っ張って、大きくしたり小さくしたりします。
* **回転（Rotate）**: 線をなぞって、パーツの向きを変えます。

## 4. 【超重要】「アンカー」を忘れずに！
空中に浮かせたブロックは、そのままだと重力で地面に落ちてしまいます。
* **アンカー（Anchor）**: ブロックを空中に「固定」するピン留めのような機能です。これ忘れると、ゲーム開始と同時にコースが崩壊します！

## 5. ブロックに魔法をかけよう（プロパティ設定）
スクリプトを書かなくても、設定を変えるだけで面白い仕掛けが作れます。
* **色と素材**: ブロックを赤くしたり、ネオンのように光らせたりします。
* **透明度（Transparency）**: 透明な「見えない床」を作ります。
* **衝突（CanCollide）**: 壁なのにすり抜けられる「隠し通路」を作ります。

## 6. チーム開発のお約束
みんなで一つの世界を作るときの大事なルールです。
* **人の作品を消さない**: 友達が頑張って作ったものを壊さないようにしましょう。
* **担当エリアを守る**: 自分の場所を決めて、そこで作業しましょう。

## 7. テストプレイとデバッグ
作ったら必ず自分で遊んでみます。
* **プレイ（Play）**: 実際にキャラクターを動かしてみます。
* **デバッグ**: 「ジャンプが届かない！」「簡単すぎる！」などの問題を見つけて、調整することを「デバッグ」と言います。何度も直して完璧なコースにしましょう。
